# ✈️ British Airways Passenger Analysis

## 📌 Project Overview
The British Airways Passenger Analysis project aims to analyze passenger reviews and satisfaction levels to understand the factors influencing the overall customer experience. The project uses Exploratory Data Analysis (EDA) and interactive dashboards to identify trends in passenger ratings, service quality, traveler preferences, and recommendation rates.

The analysis provides actionable insights that can help improve customer satisfaction and support data-driven decision-making.

---

## 🎯 Objectives
- Analyze passenger satisfaction based on reviews and ratings.
- Identify the highest and lowest rated airline services.
- Compare ratings across cabin classes and traveler types.
- Analyze recommendation percentages.
- Discover trends in customer feedback over time.
- Provide business recommendations to improve passenger experience.

## 📈 Key Insights
- Passenger satisfaction varies across different cabin classes.
- Cabin staff service received higher ratings compared to some other services.
- Seat comfort and food quality showed opportunities for improvement.
- Recommendation rates were strongly associated with overall ratings.
- Different traveler types displayed different satisfaction levels.

---

## 📌 Business Recommendations
- Improve seat comfort on long-haul flights.
- Enhance food and beverage quality.
- Maintain high standards of cabin crew service.
- Focus on customer feedback to improve low-rated services.
- Personalize services for different traveler segments.

---

## 📄 Project Outcome
This project demonstrates how data analytics can transform customer review data into meaningful business insights. It highlights the use of Power BI to perform data analysis and build interactive dashboards for better decision-making.
